# Workshop 1 Guide: "Junior Hacker: Sort Mission" (Ages 6-9)

## Facilitator Section

### Learning Objectives
- Understand what classification means
- Learn that AI sorts things by looking at features
- Practice pattern recognition

### Session Timing (~45 min)
- **Introduction:** 5 min — Welcome, enter name, explain the mission
- **Activity:** 30 min — 3 sorting missions (easy → medium → hard)
- **Discussion:** 10 min — What did you learn? How does AI do this?

### Setup
Just open `workshop-1.html` in any web browser. No installation needed.

### Discussion Prompts
- "Which mission was hardest? Why?"
- "Did any items seem like they could go in two groups?"
- "How did you decide where to put each item?"
- "AI faces the same tricky decisions — and sometimes gets it wrong!"

### Common Questions & Answers
- **"Is AI smarter than me?"** — "AI is faster at some things, but you're the one teaching it!"
- **"Can AI think?"** — "AI finds patterns, but it doesn't think like you do."

### Tips
- Help younger kids with drag & drop on small screens
- Encourage discussion between missions
- The mute button is top-right if sound is distracting

## For Developers & Engineers

### Introduction
Welcome — you're the technical mentor helping kids discover AI. Kids will sort items and learn classification. Your job: assist with tech issues, guide discussions, spark curiosity.

### Tools Used
- Pure HTML5 + CSS + JavaScript (no libraries)
- HTML5 Drag & Drop API with touch fallback
- CSS animations for celebrations
- No server, no database — 100% browser-based

### Genius Project Examples (inspire the kids)
- **Google Photos:** auto-sorts photos by people, places, things
- **Self-driving cars:** recognizing stop signs, pedestrians, lane lines
- **Recycling robots:** AI sorting plastic, glass, metal on conveyor belts
- **Mars rover:** classifying rock types on Mars
- **Airport security:** AI classifying objects in luggage
