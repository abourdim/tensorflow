# Workshop 3 Guide: "AI Engineer: Build & Deploy" (Ages 14+)

## Facilitator Section

### Learning Objectives
- Understand neural network architecture
- Write/modify real TensorFlow.js code
- Train on digit data, understand loss/accuracy/epochs
- Experiment with hyperparameters

### Session Timing (~60 min)
- **Build:** 15 min — Explore the code editor, build model architecture
- **Train:** 20 min — Adjust hyperparameters, watch training
- **Test & Experiment:** 20 min — Draw digits, test accuracy, try challenges
- **Discussion:** 5 min — Ethics, next steps

### Setup
Modern browser with good CPU. Chrome or Edge recommended.

### Tips
- Let teens experiment — don't give answers, ask questions
- "What happens if you change the learning rate to 0.1?"
- "Why is validation accuracy different from training accuracy?"
