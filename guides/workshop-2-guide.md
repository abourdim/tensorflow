# Workshop 2 Guide: "Code Breaker: Train the AI" (Ages 10-13)

## Facilitator Section

### Learning Objectives
- Understand what training data is
- See how a neural network learns from examples
- Experience the train → test → improve cycle
- Understand accuracy and why more data = better AI

### Session Timing (~45 min)
- **Briefings:** 8 min — 4 intro screens explaining neural networks and training
- **Drawing & Training:** 25 min — Draw shapes, train the model, test it
- **Discussion:** 12 min — Accuracy, improvement, real-world applications

### Setup
Open `workshop-2.html` in Chrome or Edge (TensorFlow.js works best there).

### Tips
- Encourage clear, centered drawings
- If accuracy is low, add more training data
- The training takes 5-10 seconds — use this time to explain what's happening

---

## For Developers & Engineers

### Tools Used
- TensorFlow.js (CDN)
- Simple CNN (2 conv layers + dense)
- HTML5 Canvas API
- No server — trains in the browser

### Genius Project Examples
- **Quick Draw by Google:** 50 million drawings from real people
- **Handwriting recognition:** how tablets read your writing
- **Medical X-ray scanning:** AI detecting fractures
- **Plant disease detection:** farmers photograph leaves, AI identifies diseases
- **Satellite analysis:** AI classifies land use from space
