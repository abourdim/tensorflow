## v1.0.0 (2026-03-08)

### Initial Release

**Workshops:**
- Workshop 1: Junior Hacker — Sort Mission (Ages 6-9) — 3 missions, drag & drop, touch support
- Workshop 2: Code Breaker — Train the AI (Ages 10-13) — TensorFlow.js CNN, drawing canvas, AI Mistakes Gallery
- Workshop 3: AI Engineer — Build & Deploy (Ages 14+) — Code editor, model building, training dashboard, AI Mistakes Gallery
- Workshop 4: AI Decoded (Parents & Teachers) — 7 chapters, interactive demos, ethics scenarios, AI Mistakes for adults

**Tools:**
- Tools of the Trade — TensorFlow & alternatives, 4 age-specific tabs
- Hardware Lab — 4 projects (gesture classifier, sensors, plant monitor, edge AI)
- Certificates — Generate & print with Islamic geometric borders
- Dashboard — Badge tracker with skills and recommendations
- Journal — Prompt-based learning reflections
- Time Capsule — Sealed AI predictions with countdown
- QR Posters — Printable workshop posters

**Infrastructure:**
- All-in-one single-file app (`ai-hacker-lab-single.html`)
- Hub launcher with iframe loading (`ai-hacker-lab.html`)
- All files standalone — open any directly
- Full documentation: README, FAQ, HOWTO, DEMO, DELIVERY, CHANGELOG
- 4 facilitator guides with developer sections
- Project plan v1.4

**Design:**
- Bismillah header on every page
- Dark/Light theme toggle
- Sound effects with mute toggle
- Islamic geometric patterns
- Touch-friendly mobile support
- No music, no dance, culturally respectful
- Diverse names, inclusive for all backgrounds
